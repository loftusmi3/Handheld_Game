# Handheld_Game
Handheld Raspberry Pi game device
2018-2019

Developed with a Raspberry Pi

Calibrated code for the ultrasonic sensor’s sensitivity

Coded gameplay in Python: player waves hand in front of the ultrasonic sensor to bounce the ball high enough to prevent it from hitting the “ground”

Gameplay demo available here: https://www.youtube.com/watch?app=desktop&v=5FmtEzWkOlw&feature=youtu.be
